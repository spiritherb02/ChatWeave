          📖 使用说明

          最常用流程：将 DeepSeek 对话迁移至 RikkaHub

          1. 导出 DeepSeek 聊天记录：打开 chat.deepseek.com 登录，右下角"三个点" → 系统设置 → 数据管理 → 点击"导出所有历史对话"右侧的导出按钮，获取 ZIP 文件。
          2. 准备 RikkaHub 备份：打开 RikkaHub App，左上角菜单 → 设置 → 数据备份 → 导入和导出 → 导出为文件，保存 .rikkabackup 文件。
            如果没有 RikkaHub 或想从零开始，可跳过此步，直接使用下方基准备份。
          3. 回到 ChatWeave：来源选择 DeepSeek 数据，上传第 1 步的 ZIP；在对话工作台中勾选要迁移的对话；再上传第 2 步的 RikkaHub 备份，点击合并聊天记录 → 下载合并后的 ZIP。
          4. 导入 RikkaHub：App 内 → 设置 → 数据备份 → 导入和导出 → 备份文件导入，选择合并后的 ZIP。导入后重启 App 即可看到迁移的对话。

          首次使用？下载基准备份：
          如果你还没有用过 RikkaHub，或想干净地从头开始：下载下方基准备份文件夹中与你 RikkaHub 版本对应的文件，把它当做第 2 步的"原始备份"，直接合并 DeepSeek 对话即可。
          下载入口：https://www.123865.com/s/g4IITd-1GKX
          注意：请选择与 App 版本号匹配的备份文件；高版本 App 可兼容低版本备份，反之可能失败。

          想在 RikkaHub 中继续使用 DeepSeek？
          在 RikkaHub 的"提供商"页面添加以下推理服务，两个入口均支持 DeepSeek 模型，价格通常比官方直连更实惠：
          入口 A：https://yunwu.ai/register?aff=PMO3（支持 DeepSeek + 海外大模型）
          入口 B：https://yunwu.ai/register?aff=ac4t（同上，独立推广链接）
          以上入口为推广信息，具体价格以平台实时展示为准。

          📚 更多方法

          ChatGPT 官方导出 → RikkaHub

          ChatGPT → RikkaHub

          1. 在 ChatGPT 设置中导出完整数据 ZIP。
          2. 本工具来源选"ChatGPT 数据"，上传 ZIP，勾选对话。
          3. 上传 RikkaHub 备份，点击合并并下载，导入 RikkaHub。

          YourAIScroll / ChatWeave JSON + Markdown 粘贴 → RikkaHub

          1. JSON：直接上传 YourAIScroll 或 ChatWeave 导出的 JSON 文件。
          2. Markdown：点"粘贴 Markdown 文本"，每次粘贴一个对话；建议保留 [USER] / [ASSISTANT] 标记。
          3. 进入工作台后可继续追贴更多对话，再统一合并。

          Qwen 官方导出 → RikkaHub

          1. 打开 chat.qwen.ai/settings/chats 导出 chat-export-*.json。
          2. 来源选"Qwen 官方导出 JSON"，上传即可。
          3. 国内版 qianwen.com 暂无导出入口，不在适配范围。

          多来源混合合并

          1. 依次导入不同来源的导出文件到同一工作台。
          2. 筛选、编辑、勾选你要迁移的对话。
          3. 一次合并到同一个 RikkaHub 备份，统一管理。

          提示：合并前务必保留 RikkaHub 原始备份。任何时候出问题，恢复备份即可回退。

          📋 支持列表明细

          ✅

          已支持

          • ChatGPT 官方导出 ZIP：对话文本、图片（用户上传 + 生成图）、标题与创建/更新时间
          • DeepSeek 新版导出 ZIP：完整对话树（REQUEST / THINK / RESPONSE / SEARCH / TOOL_*）、工具调用过程与搜索结果、标题与时间、重新生成的备选回复
          • DeepSeek 旧版导出 ZIP（chat.csv / coder.csv）：角色、文本、消息时间与标题
          • Grok 官方导出 ZIP（prod-grok-backend.json）：会话标题、消息内容、时间与角色
          • Qwen 官方导出 JSON（仅 chat.qwen.ai）：标题、时间戳、回答正文、深度思考摘要、联网搜索摘要、工具调用信息
          • YourAIScroll / ChatWeave JSON：URL 平台识别、角色标记、Markdown 图片/链接、深度思考标记，可保留时间戳
          • Markdown 粘贴导入：支持常见角色标记和深度思考标记，不含原始时间戳
          • 统一对话工作台：多来源并存、对话/消息级勾选、逐条编辑、合并输出为 RikkaHub 备份包
          ⚠️

          不支持 / 已知限制

          • ChatGPT：账号信息、收藏/归档状态、插件/Gizmo、群聊、消息反馈、Sora 视频、购物数据
          • DeepSeek：无法区分模型模式（快速/专家/识图）；思考时长不可用；工具调用的搜索词和链接已清空；语音消息、文件附件不支持
          • Qwen：国内版通义千问（qianwen.com）无对话导出入口
          • 通用限制：消息编辑历史、已删除消息、实时对话状态、精确 token 用量均不在导出范围内

          ChatWeave

          v1.8.0 · 跨平台 AI 聊天记录迁移工具

          ChatWeave 是一个浏览器端的对话迁移工具：把 ChatGPT、DeepSeek、Grok、Qwen 等平台的聊天记录导出后，整理、编辑、合并为 RikkaHub 可导入的备份包。

          所有数据处理在本地浏览器完成，不上传任何服务器。
